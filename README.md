Sorting Algorithm Visualizer
📌 Overview
An interactive web-based Sorting Algorithm Visualizer built with vanilla HTML, CSS, and JavaScript. Watch sorting algorithms come to life with real tile movements, sound effects, and real-time statistics!

✨ Features
5 Sorting Algorithms: Bubble, Selection, Insertion, Merge, and Quick Sort

Real Tile Movement: Tiles physically swap positions with smooth animations

Live Statistics: Track steps, comparisons, and swaps in real-time

Audio Feedback: Distinct sounds for compare, swap, and sorted actions

Algorithm Details: View time complexity, space complexity, and code

Dark/Light Theme: Toggle between themes for comfortable viewing

Responsive Design: Works seamlessly on desktop, tablet, and mobile

🛠️ Technologies
HTML5 – Semantic structure

CSS3 – Custom properties, Flexbox, animations, glassmorphism

JavaScript (ES6+) – Async/await, generators, DOM manipulation, Web Audio API

🎯 How It Works
Generate a random array of tiles

Select an algorithm from the dropdown

Adjust array size and animation speed

Click Sort to start visualization

Watch tiles move, compare, and swap in real-time

Monitor statistics and complexity in the info panel

📊 Algorithms Supported
Algorithm	Best	Average	Worst	Space
Bubble Sort	O(n)	O(n²)	O(n²)	O(1)
Selection Sort	O(n²)	O(n²)	O(n²)	O(1)
Insertion Sort	O(n)	O(n²)	O(n²)	O(1)
Merge Sort	O(n log n)	O(n log n)	O(n log n)	O(n)
Quick Sort	O(n log n)	O(n log n)	O(n²)	O(log n)
🎨 Color Legend
🟦 Default – Unsorted elements

🟨 Comparing – Elements being compared

🟪 Moving/Swapping – Elements being exchanged

🟩 Sorted – Elements in final position

Controls:

Size Slider – Change number of elements (5-20)

Speed Slider – Control animation speed (1-100)

Generate – Create new random array

Sort – Start visualization

Stop – Pause execution

🎵 Sound Features
Compare – 300Hz tone (60ms)

Swap – 600Hz tone (100ms)

Sorted – 880Hz tone (80ms)

Toggle – Sound on/off button
